Color Properties

Properties Value Description
background-color RGB, HSL, #, Color make background-color color
color RGB, HSL, #, Color make foreground-color/Text Color

Display & Postioning

Properties Value Description
position relative relative to it's default static position on the webpage
fixed specific position on the page regardless of user scrolling
absolute will ignore the element & act like it's not present on the flow of the page
sticky stick to specified position
z-index control how far back & forward the element
